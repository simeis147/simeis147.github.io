# 什么是HTML ?
<!-- more -->

::: tip HyperText Markup Language，超文本标记语言

- 超文本：超越了文本的限制，比普通文本更强大。除了文字信息，还可以定义图片、音频、视频等内容。

- 标记语言：由标签构成的语言
  - HTML标签都是预定义好的。  
    ::: tip
    \<h1> 标签展示标题  
    \<a>展示超链接  
    \<img>展示图片  
    \<video>展示视频
    :::
  - HTML代码直接在浏览器中运行，HTML标签由浏览器解析。

:::
