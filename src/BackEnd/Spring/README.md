# Spring
<!-- more -->

## 简化开发

Spring框架中提供了两个大的核心技术，分别是: IOC、AOP（事务处理）

::: tip IOC AOP

1. Spring的简化操作都是基于这两块内容

2. 事务处理属于Spring中AOP的具体应用，可以简化项目中的事务管理

:::

## 框架整合

Spring在框架整合这块已经做到了极致，它可以整合市面上几乎所有主流框架，比如:

* **MyBatis**
* MyBatis-plus
* Struts
* Struts2
* Hibernate
* ……

::: info 综上所述，对于Spring的学习，主要学习四块内容:
(1)IOC  

(2)整合Mybatis(IOC的具体应用)  

(3)AOP  

(4)声明式事务(AOP的具体应用)  

:::
